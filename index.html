Projects

Jobsite Sentry

Job-site is an advanced monitoring system. It utilizes advanced AI technology in cameras to ensure the highest level of security and surveillance on job sites. It helps the user to monitor the movement instead of watching live streaming every time. If any movement is caught by the job-site advanced AI technology, it generates an alert to the monitoring team. So, they can watch the alert and check the priority of the alert whether it is threaded or not.

Group Caption

The web-based React project that we are developing is designed to provide users with a highly customizable workspace where they can manage their personal or group projects with ease. With this platform, users can create to-do lists, organize tasks, and access project summaries, among other features. Furthermore, users have the ability to customize their workspace by changing the background color and adjusting the width of the website. Our project also includes a feature that allows users to view any available opportunities within their company.

This Car

I have designed a project for a car website that allows users to search for their desired car based on their specific requirements. Users can easily search for a car by model or make body type, and price range. Additionally, if the user is unsure about what they are looking for, they can consult a personal free shopper who can help them choose the best car for their needs. To make it easier for users to browse through the car catalog, I have implemented a slider using Swiper. This slider will display all the car catalogs to the user in an organized and user-friendly way.

Gold Leash K9

I worked on a website named "Gold Leash K9," formerly known as "My Buddy Place." This platform serves as a hub where users can discover canine companions and receive professional training services for their beloved pets. In essence, Gold Leash K9 provides a comprehensive repository of essential information about dogs, offering users the opportunity to not only find suitable canine partners but also to effectively train their own dogs.

Tailwinds Flight School Management

I have developed a web dashboard named "Tailwinds Flight School Management" for a flight school dedicated to teaching aviation enthusiasts how to pilot aircraft. This comprehensive project facilitates users in reserving their classes, while instructors can access information about their reservations, past flights, and transactions. The dashboard also offers detailed insights into the available aircraft fleet and enables users to conveniently schedule their reservations. Moreover, the platform encompasses a range of courses designed to enrich students' knowledge about different aspects of aviation and aircraft.

Management Information System (MIS)

In this project, I was responsible for working on Management Information System (MIS) functionalities. My tasks included designing various components such as password screens, modals, and multiple pages. The project itself is built using the Angular framework and Material UI for the user interface. My contributions encompassed creating visually appealing and user-friendly interfaces to enhance the overall user experience within the MIS.

WhatAPitch

In this project, I was responsible for designing new pages of WhatAPitch dashboard. My tasks included designing various components such as Billing screens, modals, and multiple pages. The project itself is built using the React for the user interface.

Taleemly (LMS)

In this project, my role involved designing the entire LSM system. I was tasked with creating a landing page featuring a custom slider that responds to mouse clicks and movement. Additionally, I designed multiple screens for the project, including the course screen, which featured two main categories: new courses and continuing courses. Furthermore, I was responsible for designing various authentication screens and modal windows as part of the project.

Space
